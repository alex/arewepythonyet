Are we Python yet?

No, not really – but we're working on it!

This site tracks performance metrics for PyPy.js, an in-browser version of the PyPy python interpreter.

There is currently no permanent build infrastructure. Benchmarks are run by hand and results are stored in static json files for offline processing. You can grab the code from github if you'd like to run them locally.
